
WB TechLand – Site Profissional

Este pacote contém um site multi-páginas pronto para publicar (Início, Serviços, Portfólio e Contato).

Como publicar gratuitamente
---------------------------
Netlify
1) Crie conta em netlify.com
2) Add new site → Deploy manually
3) Arraste a pasta inteira ou o arquivo wbtechland-site.zip

Vercel
1) Suba esta pasta para um repositório no GitHub
2) No vercel.com → New Project → Import Git Repository
3) Deploy com padrão

GitHub Pages
1) Crie um repositório e envie todos os arquivos
2) Settings → Pages → Branch main e pasta root
3) Aguarde o link do site
